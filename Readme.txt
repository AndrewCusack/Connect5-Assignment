CONNECT5 ASSIGNMENT by Andrew Cusack

This assignment is a server-client game of connect5 created using Node.js
and Java. To keep things simple for testing the game is run over the localhost 
and manages a single game instance.Confirmed working on Windows 10
using latest Java and Node.js. Follow the running instructions to begin and 
follow the onscreen instructions after that point.

Requirements: Node.js, Java set up on the pc

Running Instructions:
1. Open the terminal and run "node Connect5Server.js" to start the server
2. Open two instances of Connect5Client.java to simulate two clients connecting
3. Enjoy the game!

Alternatively running start.bat will start the server and two instances of clients.

Type "exit" at any time to terminate the session. Otherwise the other controls are
self explanatory.
